üìö Come funziona

Questa applicazione dimostra l'uso di JWT per l'autenticazione:

  1. Registrati con username e password
  2. Effettua il login per ricevere un JWT token
  3. Usa il token per accedere alle API protette dei todo
  4. Il token scade dopo 24 ore

üîå API Endpoints

POST /register - Registrazione nuovo utente
POST /login - Login e generazione JWT
GET /api/todos - Lista todo (autenticato)
POST /api/todos - Crea todo (autenticato)
PUT /api/todos/<id> - Modifica todo (autenticato)
DELETE /api/todos/<id> - Elimina todo (autenticato)
